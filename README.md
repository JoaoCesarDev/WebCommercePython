Este projeto tem como foco a construção de um site de comércio eletrônico utilizando o framework web Django. O site permite que os usuários naveguem pelos produtos, adicionem itens ao carrinho e façam pedidos através de um sistema de compras online.

Exibe uma lista de produtos disponíveis para que os usuários possam navegar e selecionar.
Oferece um carrinho de compras onde os usuários podem adicionar, remover e atualizar as quantidades de produtos.
Converte os itens do carrinho em um pedido durante o processo de finalização da compra e oferece suporte ao processamento de pagamentos.
Utiliza IDs de login e senhas para garantir acesso seguro apenas para usuários autorizados.
Abrange as principais funcionalidades de comércio eletrônico, com possibilidade de expansão e melhorias futuras.
